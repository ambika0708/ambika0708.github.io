Works

Automatic Obstacle detection system using arduino IDE with ultrasonic Sensor and LCD

1. Helps to Detect the distance between the sensor and the obstacle (object)
2. This system mainly uses an ultrasound sensor, buzzer and LED
3. When the ultrasonic sensor detects an obstacle at a distance < 3cm, the Arduino (ATmega328) orders the buzzer to ring and the red LED to light up

AI based Cryptomining for Traditional work station System
1.  Embedded Artificial Intelligence hardware board (LSAI48266x) which helps to start and stop the Crypto mining process automatically
2. Detecting the person availability near the workstation using IR sensor, If person is not available near mining PC system, AI board Automatically starts the mining process
3. Detecting the systems temperature to avoid overheating, Whenever the temperature reaches above 50 deg Celsius, AI board stops the crypto mining process, temperature is compensated immediately using cooling pad
